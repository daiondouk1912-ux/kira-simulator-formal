# kirA 外構費用シミュレーター v12

## 更新内容

v12 は、v11.1 のセッション制御・特典案内・途中通知を維持したまま、外構一式で抜けやすい追加項目を増やした版です。

### 追加項目

- タイルデッキ
- アプローチ・園路
- タイル・石貼り系アプローチ
- 土留め・高低差調整
  - 低めのブロック土留め
  - 型枠ブロック土留め
  - 現場打ちコンクリート土留め・擁壁
  - よく分からないので相談したい
- 見切り材・境界処理
- 照明・ライトアップ
- 排水・桝まわり調整
- 門柱・ポストまわり

## 金額設計の方針

- 小規模工事は安く出しすぎず、現実的な最低レンジを反映
- ただし高く見えすぎないよう、小規模でも相談しやすい金額感に調整
- 最低施工金額が反映された時だけ、結果画面に短い補足を表示
- 土留めは普通ブロック系・型枠ブロック・現場打ちを別扱い
