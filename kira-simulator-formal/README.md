# kirA 外構概算シミュレーター 正式版たたき台

作成日: 2026-03-20 JST

## この版で反映したこと
- 試作版URLの流れをベースに新規実装
- 結果画面は「合計」より「工事項目別」を主役化
- フェンス系は試作版の入力構成を維持
  - 種類
  - 設置方法
  - 長さ
- 内部ロジックで以下へ自動分岐
  - メッシュフェンス（通常新設）
  - メッシュフェンス（既存ブロック上・コア抜きあり）
  - メッシュフェンス（既存上1段積み）
- LINE通知を2回実装
  - 開始時
  - 結果表示時
- 小面積・短距離では最低施工価格を優先

## 配置
- `index.html` 画面本体
- `styles.css` スタイル
- `app.js` 画面ロジック・概算ロジック
- `netlify/functions/notify-start.js` 開始通知
- `netlify/functions/notify-result.js` 結果通知
- `netlify.toml` Netlify設定

## Netlify設定
Netlifyの環境変数に以下を設定してください。

- `LINE_WEBHOOK_URL`

これはLINE公式アカウント側、もしくは中継Webhook側の受け口URLです。

## 補足
この実装は「試作版ソースなし」での新規再構築版です。
見た目はURL上の試作版を参考にしていますが、完全一致ではなく、
正式版の仕様を優先して調整しています。
