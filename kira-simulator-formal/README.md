# kirA 外構費用シミュレーター v10

## 更新内容

v10 は、v9 の安定版をベースに「途中通知」と「その他工事入力」を改善した版です。

- 工事選択で「次へ」を押した時に、工事項目選択通知を送信
- 内容入力で「内容を確認する」を押した時に、入力完了通知を送信
- 結果画面で「この内容で相談する」を押した時に、相談ボタン通知を送信
- 「一覧にない工事も相談」を「その他の工事・気になる内容」に変更
- その他入力欄の文言を軽くして、相談前でも入力しやすく調整
- 受付番号・エリア・入力内容・概算を通知に反映
- 施工予定地エリア、受付番号、㎡/縦×横/目安選択、LINE相談導線は v9 から維持

## 更新後の確認

Netlify の最新デプロイが Published になってから、実機で以下を確認してください。

1. 工事選択で「次へ」を押すと、工事項目選択通知が届く
2. 内容入力後に「内容を確認する」を押すと、入力完了通知が届く
3. 結果画面に進むと、従来通り結果通知が届く
4. 「この内容で相談する」を押すと、相談ボタン通知が届いてLINEトークに戻る
5. その他の工事・気になる内容の入力が通知に出る
