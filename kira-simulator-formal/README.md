# KirA 外構費用シミュレーター v13.6

## 更新内容

v13.6 では、v13.1〜v13.5 の内容を含んだうえで、結果画面に「LINEに送って保存」機能を追加しました。

- 人工芝の目安選択文言を調整
  - 人工芝では「広めの庭・ドッグランなど」と表示
- 結果画面に「概算シミュレーターの感想」入力欄を追加
- 感想送信時に LINE へ通知
- 大きい面積用レンジを追加
  - 人工芝
  - 防草シート＋砕石敷き
  - 砕石敷き
- 人工芝と防草シート＋砕石敷きを同じ面積で選んだ場合、二重計算の可能性を画面上で案内
- 合計欄の「表示金額は概算の目安」注意書きを赤系で強調
- 入力欄・面積入力・目安選択まわりの見た目を使いやすい形へ戻し
- 結果画面に「この概算をLINEに送って保存する」ボタンを追加
- LINE保存ボタン押下時に、受付番号・エリア・項目・入力内容・概算を入れた文章をLINEへ渡す
- LINE保存ボタン押下時に、マスター側へ通知
- 相談ボタン文言を「この概算について質問する」に変更

## LINE保存機能について

お客様が「この概算をLINEに送って保存する」を押すと、LINE公式アカウントのトーク画面に概算結果の文章が入力されます。
LINEが開いた後、お客様がそのまま送信すると、トーク内に概算結果が残り、あとから見返しやすくなります。

※ ボタン押下通知は送られますが、実際にお客様がLINEで送信したかどうかは、LINEトーク側で確認してください。

## 注意

publicPriceMaster.js は公開用の概算レンジだけを管理するファイルです。
原価・人工原価・利益率などの内部情報は絶対に入れないでください。
