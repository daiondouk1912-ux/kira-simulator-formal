# kirA 外構費用シミュレーター v11

## 更新内容

v11 は、v10 の途中通知版をベースに「特典表示」と「セッション制御」を追加した版です。

- 結果画面に、シミュレーター経由相談の特典案内を追加
- セッション有効時間を30分に設定
- 30分経過後は新しい受付番号・新しいセッションとして扱う
- セッション更新時に通知済みフラグをリセット
- v10 の途中通知を維持
  - 工事選択完了通知
  - 内容入力完了通知
  - 相談ボタン押下通知
- v9 の基本改善を維持
  - 施工予定地エリア任意入力
  - 受付番号表示
  - ㎡ / 縦×横 / 目安選択

## 更新後の確認

Netlify の最新デプロイが Published になってから、実機で以下を確認してください。

1. 結果画面に特典案内が表示される
2. 「この内容で相談する」で相談通知が届き、LINEトークへ戻る
3. 30分以上経過後に操作した場合、新しい受付番号になる
4. 既存の開始通知・工事選択通知・内容入力通知・結果通知が動く
