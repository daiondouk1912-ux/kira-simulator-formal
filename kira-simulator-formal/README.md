# kirA 外構費用シミュレーター v13.3

## 更新内容

v13.3 では以下を更新しました。

- 人工芝の目安選択文言を調整
  - 「広めの駐車場・庭」ではなく、人工芝では「広めの庭・ドッグランなど」と表示
- 結果画面に「概算シミュレーターの感想」入力欄を追加
- 感想送信時に LINE へ通知
- 感想通知には受付番号・エリア・項目・概算・感想内容を表示
- 項目数が多い場合に LINE 通知文が長くなりすぎないよう、一部省略処理を追加

## 注意

publicPriceMaster.js は公開用の概算レンジだけを管理するファイルです。
原価・人工原価・利益率などの内部情報は絶対に入れないでください。
