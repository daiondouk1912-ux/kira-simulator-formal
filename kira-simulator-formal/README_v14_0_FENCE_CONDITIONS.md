# KirA 外構概算シミュレーター v14.0 フェンス条件分岐・価格調整版

## 目的

v13.9.1 セキュリティ強化後の安定状態を維持しながら、目隠しフェンスの概算を「長さだけ」から「高さ・取付方法」で分岐できるようにするアップデートです。

HP流入や電話問い合わせ前の費用確認で、H1800前後の目隠しフェンスがH1200程度の金額で安く表示されすぎることを防ぎます。

## 今回入れた内容

- 目隠しフェンスの入力に「高さの目安」を追加
- 目隠しフェンスの入力に「取付方法」を追加
- LIXILフェンスAB系を基準にした公開概算レンジへ調整
- H1200超は独立基礎での施工を基本に概算
- 結果画面・LINE保存内容・結果通知に高さと取付方法を表示
- 目隠しフェンスの注意書きを追加
- index.html に security-client.js の読み込み行を追加

## 価格レンジ

| 条件 | 概算レンジ | 最低施工目安 |
|---|---:|---:|
| ブロック上 H800〜H1000程度 | 20,000〜28,000円/m | 120,000〜180,000円 |
| ブロック上 H1200程度 | 24,000〜34,000円/m | 150,000〜220,000円 |
| 独立基礎 H1600程度 | 35,000〜48,000円/m | 250,000〜350,000円 |
| 独立基礎 H1800程度 | 42,000〜58,000円/m | 300,000〜400,000円 |
| H1800超・高めの目隠し | 55,000〜75,000円/m | 350,000〜500,000円 |

## 注意書き

目隠しフェンスは、高さ・取付方法・既存ブロックの状態・風の影響により金額が変わります。
ブロック上に設置する場合はH1200程度までを目安とし、それ以上の高さは独立基礎での施工を基本に概算します。
